Framework untuk RestAPI with mysql db 
- by M.Chandra.R
made with Node.js

cara pakai : 

Model : 
    - anda bisa membuat model dengan menggunakan perintah diterminal
      node mcr create:model nama_model
    - isi kolom yang dapat di edit dalam tabel

Controller :
    - anda bisa membuat controller dengan menggunakan perintah diterminal
      node mcr create:controller nama_controller
    - var dengan nama nama_model+s adalah model yang siap anda pakai 
    - terdapat opsi pilihan untuk menambah model jia tersedia
    - tambahkan custom function dalam controller anda 

Route : 
    - tambahkan route seperti yang ada pada contoh server menggunakan app
